# Skill: Zig Project Management

## Capabilities
- Initialize a new Zig binary/library.
- Build projects using the Zig build system.

## Procedures

### Initialization
- Command: `mkdir <name> && cd <name> && zig init`
- Note: This creates `src/main.zig` and `build.zig`.

### Building
- Command: `zig build` (Run from project root)
- Artifacts: Found in `./zig-out/bin/`
